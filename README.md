An AI-powered Progressive Web App that helps users decide what to cook using available ingredients by combining machine learning, computer vision, and intelligent cooking flows.

## Features
- Ingredient scanning using computer vision
- Personalized recipe recommendations
- Smart step-by-step cooking mode
- User preference learning

## Tech Stack
- Frontend: React.js (PWA)
- Backend: FastAPI
- ML: Scikit-learn
- CV: YOLOv8
- Database: MongoDB
